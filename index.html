Saved Profiles

Save any layout as a profile with a global keyboard shortcut. Activate with one hotkey from anywhere — Cmd+Shift+1 and your workspace is set.

Multi-Monitor

Screen picker shows all displays. Auto-detects which apps are on each monitor. Arranging Display 2 never touches Display 1.

Smart Window Sizing

Probes actual minimum window sizes for each app. Redistributes space intelligently per row and column. No clipping, no off-screen windows.
